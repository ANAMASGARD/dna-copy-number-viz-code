# DNA Copy Number Large Margin Visualization

This repository contains the code for the DNA copy number large margin visualization in the animint2 gallery.

## Source

Modified from the original [intreg.R](https://github.com/tdhock/animint/blob/master/inst/examples/intreg.R) from the animint package.

## Changes

- Updated to use `animint2` package
- Changed `list()` to `animint()` with proper title and source parameters
- Fixed syntax: moved `showSelected` and `clickSelects` from aesthetics to parameters
- Consolidated `showSelected2` into vector form of `showSelected`
